# firewall防火墙常用操作

linux的防火墙有很多种，一般为iptables或者firewall，其主要的功能就是为了安全。

本篇文档由于为个人使用，所以只记录了firewall防火墙的关于端口的一些基本使用，如需全面还请自行百度或谷歌：参考文档

//查询端口9200是否开启
firewall-cmd    --query-port=9200/tcp
//打开9300端口
firewall-cmd   --add-port=9300/tcp
//永久开启9300端口
firewall-cmd  --permanent --add-port=9300/tcp
//重新加载（无需断开）
firewall-cmd --reload
//重新加载（需断开）
firewall-cmd --complete-reload
