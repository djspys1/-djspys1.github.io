# bat使用

# bat文件可以直接执行，因此可以方便使用

以一个PHP项目为例，在项目根目录下新建run.bat文件

写入以下内容(内置服务器启动，具体需要啥自行修改)

php -S 0.0.0.0:8000

启动项目可以在console里直接输run来启动

想直接启动项目而不想启动apache，ide等，桌面新建bat文件

d:
cd moyun
npm run dev

要是还想顺带打开浏览器，可以再加一句 start 浏览器路径 url，或者在npm项目里配置即可
