# 未整理知识点

var array = ['a', 'b']; var elements = [0, 1, 2]; array.push.apply(array, elements);

项目启动后无法打开 依赖升级导致的无法打开 在下载一个项目并安装依赖，可能会由于部分依赖升级而导致项目无法打开，可以通过调试台查看一下错误信息，看看哪个依赖需要调整

本地启动启用压缩

-webkit-overflow-scrolling: touch 手指离开后继续滚动，会是的sticky fix定位在手指离开时失效，停下时出现
