# Ubuntu下使用Vi时方向键变乱码 退格键不能使用的解决方法

sudo vi /etc/vim/vimrc.tiny

将compatible改成nocompatible非兼容模式就可以解决方向键变ABCD的问题了。

接下来要解决Backspace键的问题也很简单，在刚才那句话后面再加一句：

set backspace=2

变为：

set nocompatible
set backspace=2
